# Fliktox Frontend

Next.js + Tailwind frontend for Fliktox.

## Key Routes

- `/dashboard`
- `/discover`
- `/watchlist`
- `/lists`
- `/clubs` and `/clubs/:id`
- `/admin` (visible only for allowed admin account)

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy env file:
   ```bash
   cp .env.example .env.local
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```

App runs on `http://localhost:3000` by default.
